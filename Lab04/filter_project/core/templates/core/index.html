شرح عملي لفلاتر جانغو (DTL Filters)

فلاتر النصوص

المتغير الأصلي name: {{ name }}

{{ name|upper }}: {{ name|upper }} (تحويل لأحرف كبيرة)

{{ name|lower }}: {{ name|lower }} (تحويل لأحرف صغيرة)

{{ name|capfirst }}: {{ name|capfirst }} (جعل الحرف الأول كبيرًا)

{{ bio|truncatewords:5 }}: {{ bio|truncatewords:5 }} (عرض أول 5 كلمات فقط)

فلاتر الأرقام والقوائم

المتغير الأصلي items: {{ items }}

{{ items|length }}: {{ items|length }} (حساب طول القائمة)

{{ items|slice:":2" }}: {{ items|slice:":2" }} (عرض أول عنصرين)

{{ price|floatformat:2 }}: {{ price|floatformat:2 }} (تقريب الرقم العشري لمنزلتين)

فلاتر منطقية وإضافات

{{ empty_variable|default:"لا توجد قيمة" }}: {{ empty_variable|default:"لا توجد قيمة" }} (عرض قيمة افتراضية)

{{ name|add:' is a developer' }}: {{ name|add:' is a developer' }} (إضافة نص)

{{ item_count|add:10 }}: {{ item_count|add:10 }} (إضافة رقم)

فلتر التاريخ

التاريخ الأصلي birthday: {{ birthday }}

{{ birthday|date:"d M Y" }}: {{ birthday|date:"d M Y" }} (تنسيق التاريخ)

فلاتر الأمان

المتغير الأصلي html_code: {{ html_code }} (لاحظ كيف تم عرض الكود ك نص عادي للأمان)

باستخدام فلتر |safe لعرضه كـ HTML:

{{ html_code|safe }}
